# 2026-06-30 工作日志

## 晚間会话：WebHarvest Playwright 集成修复与 CDN 反盗链突破

### 发现并修复的核心 Bug
**`spider.py` Line 56: `seen_assets = browser_urls`**
- Playwright 找到的所有 URL 被一次性填入 `seen_assets` 集合
- 随后 `try_download()` 检查 `if asset_url in seen_assets: return`
- **导致了所有 Playwright 发现的 URL 被静默跳过，一个都没有下载！**
- 修复：`seen_assets` 保持空集，Playwright URL 在 httpx client 创建后逐个 `try_download()`

### Playwright 事件绑定语法修复
- `browser.py` 中 `@page.on("response")` 装饰器语法错误 → `page.on("response", handler)` 正确用法
- 缺少 `from urllib.parse import urlparse` 导入
- 改用系统 Chrome (`channel="chrome"`) 替代单独下载 Chromium（避免 171MB 下载）

### Referer 头 + CDN 567 突破
- 添加 `Referer` header 到 httpx client → 仍被 CDN 拒绝
- 最终方案：Playwright CDN 回退下载 (`download_assets_via_playwright`)
- Playwright 打开完整浏览器上下文→导航到 CDN URL→fetch+base64 保存
- **成功下载 51 张之前 HTTP 567 的 CDN 图片！**
